# Convertidor
Conversor de temperatura y divisas en Java

Esta es una aplicación desarrollada en Java que permite convertir temperaturas entre Centígrados, Fahrenheit y Kelvin, así como convertir divisas entre varias monedas.

Características

Conversión de temperatura entre Celsius, Fahrenheit y Kelvin.
Conversión de divisas entre varias monedas, con tasas de cambio actualizadas a la fecha 04/04/23.
Interfaz fácil de usar e intuitiva.
Personalización de la configuración de conversión de temperatura y divisas.
Soporte sólo para el idioma Español.

Cómo usar la aplicación

Descargue y ejecute el archivo JAR de la aplicación.
Seleccione el tipo de conversión que desea realizar: temperatura o divisas.

Si desea convertir Temperatura, seleccione las unidades de origen y destino, ingrese el valor y haga clic en "Calcular".

Si desea convertir Divisas, seleccione las monedas de origen y destino, ingrese el monto y haga clic en "Calcular".

El resultado de la conversión se mostrará en la pantalla en un nmesaje JOptionPanel.

Configuración

Puede personalizar la configuración de conversión de temperatura y divisas en el archivo de configuración de la aplicación. Puede seleccionar las unidades y monedas que desee usar, sin ajustar las tasas de cambio para divisas específicas.
